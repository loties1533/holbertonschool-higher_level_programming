#  Holberton School - Higher Level Programming

Collection de projets réalisés dans le cadre de la formation à Holberton School Bordeaux.
Ce dépôt regroupe différents exercices et projets en Python portant sur la programmation orientée objet, les structures de données, les bases de données SQL et la création d’APIs.

---

## Description

Ce repository contient plusieurs projets visant à apprendre les concepts avancés de programmation en Python et l’interaction avec des bases de données.

Les projets couvrent notamment :

* la programmation orientée objet
* les structures de données
* la gestion des exceptions
* les tests
* les bases de données SQL
* les APIs REST

---

##  Objectifs

* Maîtriser les concepts avancés de Python
* Comprendre les bases de données relationnelles
* Développer des APIs
* Appliquer les bonnes pratiques de développement logiciel
* Utiliser Git et GitHub pour gérer les versions du code

## ⚙️ Prérequis

Avant d’exécuter ces projets, vous devez avoir installé :

* Python 3.x
* MySQL
* Git
* Un environnement Linux ou macOS

---

## ▶️ Exemple d'utilisation

Exécution d’un script Python :

```bash
python3 script.py
```

Exécution d’une requête SQL :

```bash
mysql -u root -p database_name < query.sql
```
