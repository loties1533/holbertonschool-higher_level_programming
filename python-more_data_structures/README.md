Qu’est-ce qu’un dictionnaire ?

C’est une structure de données qui stocke des **paires clé → valeur**

append. sert à ajouter un élément à la fin d’une liste.

range est utilisé pour générer une suite de nombres, souvent dans une boucle for

range(start, stop, step) :

start = début (inclus)

stop = fin (exclu)

step = pas (optionnel, par défaut 1)
