Python - Exceptions

try	Tester le code risqué
except	Capturer et gérer l’erreur
else	S’exécute si aucune exception n’est levée
finally	S’exécute toujours, pour nettoyage ou fin

1. Une **exception** est un problème qui survient *pendant l’exécution* et peut être gérée.
2. Les blocs `try`/`except` te permettent de **capturer et gérer des erreurs** pour que ton programme continue.
3. Les blocs `else` (si tout va bien) et `finally` (toujours) ajoutent encore plus de contrôle sur le comportement du code
